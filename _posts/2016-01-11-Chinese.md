## 以下是代码展示区 ##
<br>
以下展示如何使用命令来创建最简单的博文。<br><br>
可以通过 rake 创建 Post<br>

	$ rake post title="My first post"

同样使用 rake 创建 Page<br>

	$ rake page name="about.md"

然后是最基本的命令，添加所有文件，commit，push<br>

	$ git add .
	$ git commit -m "blog content"
	$ git push origin master
